## 📖 About The Project

<img src="Homeverse%20Property%20Website/assets/images/desktop.png" alt="Homeverse Desktop View" width="100%" />

<br /><br />

<img src="Homeverse%20Property%20Website/assets/images/mobile.png" alt="Homeverse Mobile View" width="100%" />

The **Homeverse Property Website** is a polished and responsive real estate UI concept designed for modern property platforms.  
It presents property listings, agent profiles, and real estate services in a structured and visually appealing manner on all screen sizes.

This project highlights your ability to create **responsive, business-focused websites**, ensuring smooth usability on both desktop and mobile devices.  
From grid-based property cards on larger screens to a mobile-optimized layout for on-the-go users, Homeverse delivers a consistent and professional experience.

Ideal for:
- Real estate agencies  
- Property listing platforms  
- Rental and housing websites  
- Responsive UI portfolio projects  
- Practice with mobile-first layouts  

---

## ✨ Features

- Fully responsive design (desktop & mobile)  
- Clean and modern real estate UI  
- Property listing cards with key details  
- Optimized mobile navigation  
- Smooth hover and transition effects  
- Agent and contact sections
